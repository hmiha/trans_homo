Trans-Homography
========

## Usage

    $ python trans-homo.py -n [imput_image] -dw [output_width] -dh [output_height]

## Requirement
Python
OpenCV 2.4
NumPy

## Installation
    
    $ git clone git@github.com:hmiha/trans_homo.git
    $ cd trans_homo
    $ python trans-homo.py -n [imput_image] -dw [output_width] -dh [output_height]
